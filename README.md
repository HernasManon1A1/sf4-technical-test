# StadLine Technical Test

### Specs
* Symfony 4
* MySQL5.7
* Docker
* Guzzle

### Mise en route

* Cloner le projet
* Faire un **composer:install**
* Faire un **php bin/console d:d:c && php bin/console d:s:c**
* Executer la commande: **docker-compose up -d**
* Se rendre sur http://localhost:8000
* Enjoy
